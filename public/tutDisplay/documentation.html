Scripts

Start

  • This is used as some parameter initialization for later use by scripts Soldier and Wizard. The script will run only once at the begining of the game. To initialize a variable named foobar, try my.foobar = ... . The Soldier and Wizard scripts can access and modify the variable by my.foobar. Note that the variable is shared among all player units.


Soldier, Wizard

  • This script is applied to the corresponding type of player's unit frequently for unit control.


Functions (Soldier, Wizard)

  • To apply function ff on an unit obecjt unit returned by function, try unit.ff().

this.getEnemyInRange()

  • Return an array of all enemy within unit’s sight range (yellow circle)

this.getAllyInRange()

  • Return an array of all ally within unit’s sight range (yellow circle)

this.attack ( target )

  • Attack target if target is within attack range. Otherwise performs moveTo(target). If no target given, automatically use return of getClosestEnemy() as target.

this.moveToXY ( [x, y] )

  • to move towards a specific point in map
  • return true if arrive (x,y)

this.moveTo ( target )

  • Move towards a target in map

this.getClosestEnemy ()

  • Return the closest enemy in map, including fortress.

this.distance ( target )

  • Return distance to target.

this.getAllEnemy ()

  • Return an array of all enemy in map, including fortress.

this.getCoor ()

  • Return unit's coordinate as [x,y]

this.getHP ()

  • Return unit's health.

this.getName ()

  • Return unit's name.
